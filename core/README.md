# core/ — 共享核心模块

所有机器人共用的底层能力，不需要单独运行。

## 文件说明

| 文件 | 做什么 |
|------|--------|
| `llm.py` | 大模型调用统一封装，支持 DeepSeek / 豆包 / Kimi，带自动重试 |
| `feishu_client.py` | 飞书 API 客户端：发消息、操作日历、创建文档 |
| `feishu_webhook.py` | 飞书群 Webhook 推送：把讨论过程实时发到飞书群 |
| `utils.py` | 工具函数：文本截断、时间戳、运行记录保存 |

## 谁在用

- **所有机器人** 都通过 `llm.py` 调用大模型
- **所有机器人** 都通过 `feishu_client.py` 和飞书交互
- **脑暴/规划** 通过 `feishu_webhook.py` 把过程推送到群
- **所有机器人** 都用 `utils.py` 的工具函数
