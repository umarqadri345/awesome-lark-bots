# 脑暴 brief：给三体卖地球卖个高价

## 主题
如何把「地球」卖给三体人、且卖个高价：形成可执行、有记忆点的体验或传播方案（可为 三体 IP 联动、事件营销、创作者/玩家向活动等）。

## 背景
三体 IP 拥有高认知与强情感连接；「卖地球」是一个有张力、可被玩坏的创意钩子。品牌或项目方希望借此做一场低成本、有传播性、能让参与者主动讲述的体验或 campaign，而不是简单联名或物料投放。

## 核心挑战
- 「卖地球」如何从梗转化为可被体验、可被传播的具体瞬间？
- 谁在「卖」、谁在「买」、参与者扮演什么角色？如何让人愿意主动参与和分享？
- 如何兼顾 IP 调性、避免过度娱乐化，同时又有足够新颖性和行为改变？

## 现实约束
- 成本可控，不依赖大规模投放
- 可落在线下、社媒、游戏内/外或混合场景
- 方案需可被团队主导落地

## 核心目标
创造一个具体、可被讲述的瞬间或体验：参与者觉得「这个我参与了」「这个值得晒」「这个我能讲给别人听」，并愿意主动传播。

## 成功标准
- 参与者能清晰说出「我做了什么」「为什么愿意分享」。
- 社媒或渠道上出现真实、可被识别的 UGC 或讨论（而非硬广）。

## 核心任务
设计一个「核心体验原子」：当【触发条件】，参与者会【经历具体事件】，导致他们【产生具体行为】（如主动分享、复购或推荐）。
