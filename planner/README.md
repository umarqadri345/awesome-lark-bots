# planner/ — 规划机器人

理性六步结构化决策：问题定义 → 现状分析 → 方案生成 → 评估矩阵 → 执行计划 → 反馈机制。

## 文件说明

| 文件 | 做什么 |
|------|--------|
| `bot.py` | 飞书长连接入口 — 接收消息、解析模式、启动规划流程 |
| `run.py` | 规划主引擎 — 按步骤调用 LLM 生成分析和方案 |
| `prompts.py` | 提示词库 — 定义每一步的 AI 行为和输出格式 |
| `__main__.py` | 启动入口 — `python3 -m planner` |

## 快速使用

```bash
# 飞书长连接
python3 -m planner

# 命令行直接运行
python3 -m planner.run --topic "Q3 增长策略" --mode "快速模式"
```

## 五种模式

| 模式 | 步骤 | 适合 |
|------|------|------|
| 完整规划 | 1→2→3→4→5→6 | 重大决策 |
| 快速模式 | 1→3→4→5 | 日常规划 |
| 分析模式 | 1→2 | 先看看分析 |
| 方案模式 | 3 | 只要选项 |
| 执行模式 | 5 | 已有方向，要步骤 |
